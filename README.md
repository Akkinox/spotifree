# Spotify

Este proyecto fue creado con [Angular CLI](https://github.com/angular/angular-cli) version 12.2.6.

## Descargar dependecias

Ejecuta `npm install` para bajar las dependencias del proyecto como tal

## Development server

Ejecuta `ng serve` para levantar el servidor. Navega a `http://localhost:4200/`. Aca podrás revisar el proyecto como tal.

## Build

Ejecuta `ng build` para generar el build de la aplicación, esto te generara el empaquetado de la app en la carpeta `/dist`
